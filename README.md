## How to run app:

- Clone repository:
```bash
git clone git@github.com:LenaKorsakov/camera-shop-2023.git
```

- Install dependencies repository:

```bash
npm install
```

- Run application:

```bash
npm start
```

- Check tests
```bash
npm test
```
